## Develop Reapo

Rebased to use electron-forge. To install Electron Forge:

```npm i -g electron-forge```

### To install

After cloning the repo and/or opening the reapo dir:

```npm i```

### To start

```npm run start```

### To package

```npm run package```

### To make (create an installer for your OS)

```npm run make```

### To publish

```npm run publish```
